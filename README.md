# 💳 Payments Learning Lab

An interactive training platform designed to teach the fundamentals of payment processing through visual lessons, real-world scenarios, quizzes, and hands-on learning.

Unlike traditional SOPs or static documentation, Payments Learning Lab provides a structured onboarding experience for new employees in the payments industry.

---

## 🚀 Why I Built This

After working in technical support, merchant onboarding, implementation, and partner success at multiple Independent Sales Organizations (ISOs), I noticed a common problem:

New employees were expected to learn through:

- Shadowing coworkers
- Outdated SOPs
- Internal wikis
- Trial and error

There was no modern, interactive learning platform built specifically for entry-level payment professionals.

Payments Learning Lab is my solution.
